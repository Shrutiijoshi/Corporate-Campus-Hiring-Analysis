# Corporate Campus Hiring Analysis
This project analyzes corporate campus hiring trends for fresh graduates in India.

[<img src="https://varsitypro.club/wp-content/uploads/2024/07/CAMPUS-PLACEMENT-1-1024x538.jpg" alt="report1" width="1000" height="450">](https://varsitypro.club/wp-content/uploads/2024/07/CAMPUS-PLACEMENT-1-1024x538.jpg) &nbsp;

## Project Overview :
This project analyzes corporate campus hiring trends for fresh graduates in India. The analysis includes salary distribution, job roles, industry-wise hiring trends, and job board effectiveness. The insights are visualized through an interactive Power BI dashboard.

## Tools Used :
- Power BI (Data Analysis & Visualization)
- Excel (Data Cleaning & Processing)

## Dashboard Features :
1. **Overview Page** :
- Average Salary (LPA), Total Job Roles, Minimum & Maximum Salary
- Industry-wise salary distribution
- Salary distribution histogram

2. **Performance Page** :
- Count of job roles by industry
- Average salary & bonus for companies
- CTC comparison for different job roles

3. **Job Board Source Page** :
- Average CTC by job board source
- Count of job roles by job board source
- Number of companies hiring from each job board

## Key Insights :
- **Industry Trends** : Banking & Finance, IT Services, and EdTech have the highest number of job roles.
- **Salary Distribution** : The average salary is 18.35 LPA, with a minimum of 5 LPA and a maximum of 30 LPA.
- **Highest-Paying Roles** : Software Engineers and Product Managers have the highest CTC (~23+ LPA).
- **Job Board Effectiveness** : Glassdoor and Internshala provide the highest average CTC (~23 LPA), while Monster India has the lowest (~18 LPA).
- **Company Salary Trends** : Companies like Goldman Sachs, Citi, and CRED offer the highest salaries, while others like Adobe and Freshworks have relatively lower packages.

## Conclusion :
The analysis highlights significant differences in salaries based on industries, job roles, and job boards. IT and Finance sectors dominate hiring, and job boards like Glassdoor and Internshala offer better salary prospects. The dashboard provides useful insights for fresh graduates to strategize their job search based on salary trends and industry demand.
